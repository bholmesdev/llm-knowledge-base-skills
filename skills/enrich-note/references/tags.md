# Tag registry

The canonical list of topic tags. `enrich-note` reads this before tagging.

**Rules**

- Reuse an existing tag whenever it fits. Coin a new one only when nothing matches.
- Keep the set small. Prefer a broad existing tag over a narrow new one.
- When you coin a tag, add it here with a one-line description, kebab-case, before using it.
- Topics only. Lifecycle tags (e.g. `voice-transcription`) do **not** belong here.
